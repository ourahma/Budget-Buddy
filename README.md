# Budget Buddy

Budget Buddy is a Java Enterprise Edition (JEE) application designed for personal budget management. It helps users manage their finances by tracking budgets, expenses, financial goals, and debts.

## Features

1. **Authentication**
   - Allows users to create an account and authenticate to access the application securely.

2. **Budget Management**
   - Users can create, modify, and delete budgets for different expense categories.

3. **Budget Balance Consultation**
   - Users can view the balance of their budgets after associating them with expenses.

4. **Expense Management**
   - Users can create, manage, and delete expenses. Expenses can be linked to specific budgets.

5. **Expense Planning Visualization**
   - Provides users with the ability to visualize their spending plan for the current month.

6. **Financial Goal Management**
   - Users can set, modify, or delete financial goals with deadlines, make deposits or withdrawals, and track their progress.

7. **Debt Management**
   - Allows users to create, modify, and delete debts, add funds to their debt, and monitor debt repayment progress.

8. **User Profile Management**
   - Users can manage their profile, including changing their password and logging out of the application.

## Usage

1. **Sign Up and Log In**:
   - Create an account and log in to start managing your budgets and expenses.

2. **Manage Budgets and Expenses**:
   - Use the application interface to create, modify, and delete budgets and expenses.

3. **Set and Track Financial Goals**:
   - Define your financial goals and track your progress within the application.

4. **Handle Debts**:
   - Manage your debts and monitor repayment progress.

5. **Profile Management**:
   - Update your profile information and change your password as needed.
